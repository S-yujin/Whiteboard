# Whiteboard
Python 소켓 + GUI(PyQt) 기반 공유 화이트보드 프로그램 구현

---
## 주요기능
### 1. 필수 구현 기능
* **실시간 드로잉 (Shared Drawing)**: 모든 사용자가 동일한 캔버스에 동시에 그림을 그리고 실시간으로 동기화
* **사용자 닉네임 표시 (Nickname Display)**:
    - 그림을 그리는 사용자의 닉네임이 펜 끝에 실시간으로 표시
    - 드로잉이 완료되면 해상 선 끝에 소유자 정보가 영구적으로 기록
* **채팅 기능 (Chatting)**: 드로잉과 별개로 우측 하단 채팅창을 통해 실시간 메시지를 주고 받을 수 있음
* **화면 초기화 (Clear Board)**: '전체 화면 초기화' 버튼을 통해 서버에 연결된 모든 클라이언트의 보드를 동시에 리셋

### 2. 추가 선택 기능 (구현 완료)
* **지우개 기능 (Eraser)**: 특정 부분을 배경색(흰색)으로 덮어 지우는 기능
* **이미지 저장 (Export PNG)**: 현재까지 작업한 화이트보드 내용을 PNG 이미지 파일로 저장
* **색생 및 굵기 설정**: Color Dialog와 Slider를 이용해 펜의 색상과 굵기를 자유롭게 변경 가능 

---

## 실행 방법

### 요구 사항
* Python 3.x
* PyQt5 라이브러리

### 설치 및 실행
```bash
# 1. 필수 라이브러리 설치
pip install PyQt5

# 2. 서버 실행 (Port: 5000 기반)
python server.py

# 3. 클라이언트 실행 (여러 개 실행하여 다중 접속 테스트 가능)
python client.py
